# Интерактивный слайдер для сайта

## Описание проекта

Проект представляет собой автономный интерактивный слайдер для отображения завершенных проектов строительной компании. Слайдер реализован с использованием чистого JavaScript без сторонних библиотек и готов для интеграции в рабочий сайт.

## Функциональность

### Основные возможности:

1. Переключение проектов: Отображение информации о разных проектах (город, площадь, сроки, стоимость)
2. Несколько способов навигации: 
- Навигационные кнопки (вкладки)
- Точечные индикаторы
- Стрелки "вперед/назад"
3. Циклическая навигация: Автоматический переход к первому слайду после последнего и наоборот
4. Активные состояния: Визуальное выделение текущего слайда во всех элементах управления

### Структура данных:

Каждый слайд содержит:
- Название города и района
- Площадь квартиры
- Сроки ремонта
- Стоимость (по запросу)
- Изображение проекта

## Техническая реализация

- HTML5: Семантическая разметка с улучшенной доступностью (ARIA-атрибуты)
- CSS3: Современные стили с использованием Grid и Flexbox
- Vanilla JavaScript: Чистый JS без зависимостей

## Ключевые особенности кода:

### JavaScript:

Модульная структура с четким разделением данных, состояния и логики
Единая функция рендеринга для обновления всех элементов интерфейса
Обработчики событий для всех элементов управления
Циклическая логика переключения слайдов

### CSS:

Адаптивный дизайн с мобильной версией
Плавные переходы и состояния hover
Современные подходы к именованию классов (BEM-like)
Темная тема с акцентным цветом (#e3b873)

### HTML:

Семантические теги для улучшения доступности
ARIA-атрибуты для скринридеров
Логическая структура с четким разделением контента

## Установка и использование

### Локальный запуск:

1. Склонируйте или скачайте проект
2. Откройте файл index.html в браузере
3. Для разработки можно использовать Live Server или аналогичные инструменты

### Интеграция в существующий проект:

1. Скопируйте файлы проекта в структуру вашего сайта
2. Подключите CSS и JS файлы:
```
<link rel="stylesheet" href="css/slider-style.css">
<script src="js/slider.js" defer></script>
```
3. Добавьте HTML1структуру в нужное место на странице
4. Настройте пути к изображениям и иконкам

## Настройка и кастомизация

### Добавление новых слайдов:

Отредактируйте массив SLIDER_DATA в файле slider.js:

```
const SLIDER_DATA = [
    {
        location: 'Название города<br>Район',
        size: 'Площадь',
        duration: 'Сроки',
        price: 'Стоимость',
        picture: 'путь/к/изображению.jpg'
    },
    // ... дополнительные слайды
];
```
### Изменение стилей:

#### Основные настраиваемые параметры в style.css:

- Цвета темы: #131313 (фон), #e3b873 (акцент)
- Размеры контейнера: max-width: 1200px
- Отступы и промежутки
- Адаптивные брейкпоинты

#### Точки адаптивности:

- Десктоп: > 900px
- Мобильные устройства: ≤ 900px (вертикальное расположение)
- Особенности реализации

#### Доступность (a11y):

- Поддержка навигации с клавиатуры
- ARIA-атрибуты для скринридеров
- Правильные семантические теги
- Контрастные цвета для лучшей читаемости

#### Производительность:
- Оптимизированная перерисовка DOM
- Минимальное количество обработчиков событий
- Эффективное использование CSS-переходов

#### Кроссбраузерность:

- Поддержка современных браузеров
- Graceful degradation для устаревших
